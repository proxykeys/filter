# ProxyKeys Filter Lists

Automatically updated antifilter lists in multiple formats for all proxy clients.

## Lists Sources

- [antifilter.download](https://antifilter.download/)
- [community.antifilter.download](https://community.antifilter.download/)
- [Loyalsoldier/surge-rules](https://github.com/Loyalsoldier/surge-rules)

## Update Schedule

Lists are updated automatically every 6 hours via GitHub Actions.

## Available Formats

| Format | File | Clients |
|--------|-------|----------|
| **Surge/Clash/Mihomo** | `surge-all-domains.txt` | Mihomo, Clash, Surge |
| | `ipcidr-all.txt` | Mihomo, Clash, Surge |
| **Shadowrocket** | `shadowrocket-all-domains.txt` | Shadowrocket, Quantumult X, Loon, Stash |
| | `shadowrocket-all-ip.txt` | Shadowrocket, Quantumult X, Loon, Stash |
| **Sing-box** | `singbox-domains.json` | Sing-box |
| | `singbox-ip.json` | Sing-box |
| **DNSMasq** | `dnsmasq-domains.conf` | OpenWrt, routers |
| **AdGuard** | `adguard-domains.txt` | AdGuard Home |
| **Pi-Hole** | `pihole-domains.txt` | Pi-Hole |
| **HOSTS** | `hosts-domains.txt` | Windows/Linux/Mac |
